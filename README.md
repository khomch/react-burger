# **Stellar Burger: сервис заказа бургеров на React**
Репозиторий для проекта `Stellar Burger`: конструктор бургеров (drag-and-drop) с возможностью отслеживать ленту заказов в онлайне (через WebSocket). 

## Технологии
1. React
2. Redux
3. TypeScript
4. Cypress
